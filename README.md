## Instagram Dummy

Instagram dummy has the following features:
1. Listing of posts
2. Actions on each post (like comment, share, save)
3. Each post have comments
4. Stories list
5. User can see story
6. Current username and profile icon
7. Header with search, profile, messages, notifications
8. User can search by username
9. User can see notifications list
10. Call API dummy for listing of post
11. Responsive design
12. Notification

This application is already connected to the database via an API call. Receive notifications every time a user likes a post.

For back end see : [github repo](https://github.com/anantasatyadhamma/instagram-dummy-api)
